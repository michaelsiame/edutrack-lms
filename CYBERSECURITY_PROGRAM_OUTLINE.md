# Cybersecurity Certificate Program Outline (3 Months)

## Program Overview

The Cybersecurity Certificate Program equips students with the practical and theoretical skills needed to protect computer systems, networks, and data from cyber threats. Through hands-on labs, real-world case studies, and industry-aligned curriculum, the program prepares learners for entry-level cybersecurity roles and further studies in information security.

## Course Duration

**3 Months (12 Weeks)** | Approximately 240 contact hours

---

## Module 1: Foundations of Computing and Mathematics (Weeks 1-3)

### Topics

- **Computer Fundamentals** - Hardware components, system architecture, and how computers process data
- **Operating Systems** - Windows and Linux basics, file systems, user management, and command-line navigation
- **Introduction to Programming Logic** - Variables, control structures, pseudocode, and basic scripting (Python or Bash)
- **Mathematics for Cybersecurity** - Number systems (binary, decimal, hexadecimal), Boolean algebra, and set theory
- **Networking Essentials** - IP addressing, subnetting, OSI model, TCP/IP, DNS, and DHCP

### Learning Outcomes

By the end of this module, students will be able to:

1. Explain how computer hardware and software interact at a fundamental level
2. Navigate and perform basic tasks in both Windows and Linux environments
3. Convert between binary, decimal, and hexadecimal number systems
4. Apply logical thinking and basic scripting to solve simple computing problems
5. Describe how devices communicate over a network using the TCP/IP model

### Assessment

- Practical lab: OS installation and configuration
- Quiz: Number systems and networking fundamentals

---

## Module 2: Cybersecurity Principles and Defense (Weeks 4-7)

### Topics

- **Introduction to Cybersecurity** - CIA triad (Confidentiality, Integrity, Availability), defense-in-depth, and the cybersecurity landscape
- **Cyber Threats and Attack Types** - Malware, phishing, social engineering, DDoS, man-in-the-middle, SQL injection, and ransomware
- **Network Security** - Firewalls, VPNs, network segmentation, DMZs, and wireless security
- **Cryptography** - Symmetric and asymmetric encryption, hashing algorithms, digital signatures, and PKI
- **Authentication and Access Control** - Multi-factor authentication, password policies, role-based access control (RBAC), and identity management
- **Database Security** - SQL injection prevention, data encryption at rest, access controls, and backup strategies
- **Intrusion Detection and Prevention** - IDS/IPS concepts, signature-based vs. anomaly-based detection, and SNORT basics
- **Security Architecture** - Endpoint protection, patch management, and secure configuration baselines

### Learning Outcomes

By the end of this module, students will be able to:

1. Identify common cyber threats and explain how each attack vector works
2. Configure basic firewall rules and network security controls
3. Apply cryptographic concepts to protect data in transit and at rest
4. Implement strong authentication mechanisms and access control policies
5. Explain how IDS/IPS systems detect and prevent intrusions

### Assessment

- Practical lab: Firewall configuration and network traffic analysis
- Group project: Threat landscape report on a chosen industry sector

---

## Module 3: Threat Detection, Incident Response, and Governance (Weeks 8-11)

### Topics

- **Security Information and Event Management (SIEM)** - Log aggregation, correlation rules, dashboards, and alerting (using Splunk or ELK Stack)
- **Log Monitoring and Analysis** - System logs, network logs, event correlation, and identifying indicators of compromise (IoCs)
- **Vulnerability Assessment** - Scanning tools (Nessus, OpenVAS), vulnerability scoring (CVSS), and remediation prioritization
- **Threat Hunting** - Proactive detection techniques, threat intelligence feeds, and the MITRE ATT&CK framework
- **Incident Response** - IR lifecycle (Preparation, Identification, Containment, Eradication, Recovery, Lessons Learned), playbooks, and tabletop exercises
- **Digital Forensics Fundamentals** - Evidence preservation, chain of custody, disk imaging, and basic forensic analysis
- **Risk Management** - Risk assessment methodologies, risk registers, and business impact analysis
- **Cybersecurity Governance** - Policies, standards, compliance frameworks (ISO 27001, NIST), data protection regulations, and professional ethics

### Learning Outcomes

By the end of this module, students will be able to:

1. Use a SIEM tool to monitor, detect, and investigate security events
2. Conduct a basic vulnerability assessment and prioritize remediation actions
3. Execute an incident response plan following industry-standard procedures
4. Apply the MITRE ATT&CK framework to analyze adversary behavior
5. Explain key governance frameworks and their role in organizational security

### Assessment

- Practical lab: SIEM-based threat detection exercise
- Scenario-based exam: Incident response walkthrough

---

## Module 4: Capstone Project and Practical Training (Week 12)

### Hands-On Activities (Integrated Throughout and Culminating in Week 12)

- **Network Monitoring** - Packet capture and analysis with Wireshark
- **Ethical Hacking Concepts** - Reconnaissance, scanning, and exploitation basics using Kali Linux
- **System Hardening** - Securing Windows and Linux servers against common attack vectors
- **Security Tools Proficiency** - Nmap, Wireshark, Metasploit (demo), Burp Suite (intro), and Splunk/ELK
- **Cybersecurity Case Studies** - Analysis of real-world breaches (e.g., SolarWinds, Colonial Pipeline, Equifax)

### Capstone Project

Students will complete a comprehensive capstone that includes:

1. Conducting a vulnerability assessment on a simulated network
2. Detecting and analyzing a simulated security incident using SIEM
3. Writing an incident response report with findings and recommendations
4. Presenting results to a panel (simulated stakeholder briefing)

### Assessment

- Capstone project report and presentation (pass/fail with rubric)

## Career Opportunities

Graduates of this program can pursue entry-level roles such as:

- **SOC Analyst (Tier 1)** - Monitor security alerts and escalate incidents
- **Junior Security Analyst** - Assist with vulnerability assessments and compliance audits
- **Network Security Technician** - Configure and maintain network security infrastructure
- **IT Support Security Specialist** - Implement security controls in IT environments
- **Cybersecurity Assistant** - Support security teams with monitoring and documentation

---

## Certification Pathways

Upon successful completion, students will receive a **Certificate in Cybersecurity**. The program also prepares students for industry certifications:

- CompTIA Security+
- CompTIA Network+
- Cisco CyberOps Associate
- (ISC)2 Certified in Cybersecurity (CC)

---

## Target Audience

- School leavers seeking a career in technology
- College and university students looking to specialize in cybersecurity
- ICT students and IT professionals wanting to upskill
- Career changers entering the cybersecurity field
- Beginners with basic computer literacy

## Entry Requirements

- Basic computer literacy (ability to use a computer and browse the internet)
- No prior programming or cybersecurity experience required
- Access to a computer capable of running virtual machines (8GB RAM recommended)

## Delivery Mode

- In-person, online, or blended learning
- Minimum 70% attendance required
- All practical labs conducted in a safe, sandboxed virtual environment
