## Module 1: Foundations of Computing and Mathematics

### Topic 3 - Python Programming (Special Focus)

**For Instructors:**
- `Topic_3_Programming_Logic.pptx` - Main lecture slides
- `Python_Practical_Exercises.pptx` - Hands-on exercise slides (25 slides)
- `Topic_3_Python_Practical_Exercise.md` - Comprehensive written guide

**For Students:**
- Navigate to `topic3_programming/python_exercises/`
- Start with `00_cheat_sheet.py` for quick reference
- Work through exercises `01_` to `07_` for fundamentals
- Complete scenarios `08_` to `10_` for practical application
- Finish with `99_mini_project.py` as capstone exercise

### Running Python Exercises

```bash
cd course_materials/module1_foundation/topic3_programming/python_exercises
python3 01_hello_security.py
```
