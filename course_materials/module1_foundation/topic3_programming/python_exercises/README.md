# Topic 3: Python Programming Practical Exercises

This folder contains hands-on Python exercises for the **Introduction to Programming Logic** topic in Module 1 of the Cybersecurity Certificate Program.

## How to Use These Exercises

### Prerequisites
- Python 3.6 or higher installed on your computer
- A text editor (VS Code, Sublime Text, or even Notepad)
- Command line/terminal access

### Running Python Scripts

1. Open a terminal/command prompt
2. Navigate to this folder:
   ```bash
   cd topic3_python_exercises
   ```
3. Run any script:
   ```bash
   python3 01_hello_security.py
   ```
   Or on Windows:
   ```bash
   python 01_hello_security.py
   ```

## Troubleshooting

### "python3: command not found"
Try using `python` instead of `python3`:
```bash
python 01_hello_security.py
```

### Permission Denied (Linux/Mac)
Make the file executable:
```bash
chmod +x 01_hello_security.py
```

### Syntax Errors
- Check that you're using Python 3 (not Python 2)
- Make sure indentation uses spaces, not tabs
- Check for missing colons (`:`) at the end of `if`, `for`, `while`, `def` lines
